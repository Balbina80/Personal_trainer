# 💪 AJ Fitness – Personal Trainer Website

A modern, responsive website built for a personal trainer client.  
Designed to promote fitness services, showcase training sessions, and allow users to get in touch.

🌐 Live site: https://ajfitness.uk

---

## ✨ Features

- Clean and modern UI design
- Fully responsive layout (mobile-friendly)
- Hero section with background image
- Training videos section
- Contact page with form
- Instagram and social links
- Smooth navigation between pages

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- Git & GitHub
- GitHub Pages (deployment)
